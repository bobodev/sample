TIMESTAMP
 
 
 
 
Derby 还提供了 TIMESTAMP 数据类型，把 TIME 和 DATE 数据类型组合到一个类型中，表示准确的时间。
derby库中字段的命名规则：
（1）不区分大小写
 （2）最长128个字符
 （3）必须以字母开头
 （4）必须只包含Unicode字母、下划线和Unicode数字。通过把名称放在双引号中，可以避开这些规则，使用双引号允许名称区分大小写以及包含附加字符（包括空格）。

2、建表

CREATE TABLE B_UserInfo (
  id int NOT NULL generated always as identity,
  memberId int NOT NULL,
  personId int NOT NULL,
  enterpriseId int NOT NULL,
  personCode varchar(24) DEFAULT NULL,
  personName varchar(60) NOT NULL,
  isHideBirthday smallint NOT NULL DEFAULT 0,
  timeCreated timestamp DEFAULT CURRENT_TIMESTAMP,
  timeModified timestamp DEFAULT CURRENT_TIMESTAMP,
  PRIMARY KEY (id)
)

（1）“GENERATED ALWAYS AS IDENTITY” – Derby always provides auto-incremented sequence values to this column. You are not allowed to specify your own values. 由Derby负责提供自增的主键值, 该值不允许插入.  
（2）“GENERATED BY DEFAULT AS IDENTITY” – Derby provides auto-incremented sequence values to this as default only when you are not providing values. 仅在未提供主键值时, 由Derby负责插入.
